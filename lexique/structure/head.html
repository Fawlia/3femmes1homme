Balise <head>

Description

Elément structurel d'un document HTML. Contient les informations essentielles à la description du document qui sera affiché dans le corps du document :

  • Titre
  • Feuilles de styles utilisées
  • JavaScript utilisé
  • Meta pour spécifier l'auteur, l'encodage...
