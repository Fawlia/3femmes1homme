HTML

HTML (HyperTet Markup Language) est un langage descriptif utilisé pour structurer le contenu d'une page (ses textes, ses images, ses liens...)
Un document HTML est un fichier texte qui contient des balises (ou tag), ces balises doivent être utilisées d'une certaine façon pour décrire correctement la structure d'un document. Les balises indique au navigateur comment afficher le document.

La structure de base d'un document HTML est défini avec un ensemble de balises spéciales.
Ces éléments ne doivent pas apparaître plus d'une fois dans le document. On trouve dans ces balises :

  • <html>
  • <head>
  • <meta>
  • <body>
  • <title>

Description de la balise <html>

La balise HTML permet d'indiquer au navigateur web que le document auquel il accède est un document HTML. C'est l'élément racine de la page.
